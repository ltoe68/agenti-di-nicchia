# Acme SaaS - Knowledge Base

## Pricing
- The Basic plan costs $29/month.
- The Pro plan costs $99/month and includes API access.
- The Enterprise plan is custom pricing.

## Account Management
- To reset your password, go to Settings > Security > Reset Password.
- You can delete your account by emailing support@acme.com.

## API Usage
- The API rate limit is 100 requests per minute for Pro users.
- You can find your API key in the Developer Dashboard.
